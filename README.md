# Space Invaders Game

A classic Space Invaders game developed using Python and Pygame.

## Features

* Player spaceship movement
* Enemy spawning and movement
* Bullet shooting system
* Collision detection
* Score tracking
* Sound effects and background music
* Game Over and Restart functionality

## Controls

* Left Arrow → Move Left
* Right Arrow → Move Right
* Spacebar → Shoot

## Technologies Used

* Python 3.12
* Pygame 2.6.0

## Installation

1. Clone the repository

git clone https://github.com/deoregayatri/Space_Invaders_Game.git

2. Install pygame

pip install pygame

3. Run the game

python Game.py
